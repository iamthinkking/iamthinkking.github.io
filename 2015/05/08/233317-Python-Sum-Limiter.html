Python: Sum Limiter

08 May 2015

This script tests simple sumation limits you give it. It repetitively adds your first value until it is the maximum value it can be, to be as close as possible to your second value.

def sumlimiter(x,y):
   adder= x
   def brain(adder,x,y):
      if adder > y:
         return "you went beyond your limit"
      elif (y-adder) < x:
         return adder
      else:
         return brain(adder+x,x,y)
   return brain(adder,x,y)
